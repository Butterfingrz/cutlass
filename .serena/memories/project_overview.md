# CUTLASS 项目概述

## 项目简介
CUTLASS (CUDA Templates for Linear Algebra Subroutines and Solvers) 是 NVIDIA 开发的高性能矩阵乘法(GEMM)和相关计算的 C++ 模板库。

## 版本信息
- **当前版本**: 4.2.0
- **发布时间**: 2025年9月
- **许可证**: BSD-3-Clause

## 项目目标
- 提供高性能的 CUDA C++ 模板抽象，用于线性代数计算
- 支持分层分解和数据移动策略
- 将"移动部件"分解为可重用的模块化软件组件和抽象
- 通过自定义平铺大小、数据类型和算法策略进行专业化和调优
- 作为自定义内核和应用程序中的构建块

## 核心特性
1. **广泛的数据类型支持**:
   - 混合精度计算
   - FP64, FP32, TF32, FP16, BF16
   - 8位浮点类型 (e5m2, e4m3)
   - 块缩放数据类型 (NVIDIA NVFP4, OCP 标准 MXFP4/6/8)
   - 窄整数类型 (4位和8位有符号/无符号)
   - 二进制1位数据类型

2. **架构支持**:
   - Volta, Turing, Ampere, Ada, Hopper, Blackwell 架构
   - 从 Volta (计算能力 7.0) 开始支持

3. **CUTLASS DSLs**:
   - Python 原生接口用于编写高性能 CUDA 内核
   - CuTe DSL: 低级编程模型，与 CuTe C++ 抽象完全一致
   - 无性能妥协的内核编写
   - 更平滑的学习曲线和更快的编译时间

## 主要组件
- **CUTLASS C++ 模板库**: 头文件库，包含线性代数的核心抽象
- **CuTe**: 描述和操作线程和数据张量的核心库
- **示例**: 92+ 个示例展示各种用法场景
- **工具**: 包括性能分析器和实用程序
- **测试**: 基于 GTest 的完整单元测试套件

## 目标用户
- 学生、研究人员和性能工程师
- 需要高性能线性代数计算的开发者
- CUDA 内核开发人员
- 深度学习框架开发者
